# Portfolio Web Page

Welcome to the **Portfolio Web Page** project! This repository contains the HTML code for my portfolio, showcasing my skills and work experience. 

## Repository Link
[GitHub Repository](https://github.com/CharlotteMwenda/gomycode-html-portfolio-project)

## Project Overview

This project is part of my second checkpoint in the learning process. The goal is to create a portfolio web page using HTML, inspired by a provided example. The web page will highlight my work, provide information about me, and enable visitors to contact me.

## Features

- **Semantic HTML Tags**: Utilizing semantic tags to enhance the structure and accessibility of the web page.
- **Multimedia Integration**: Incorporating video/audio elements to enrich the user experience.
- **Local Links, Lists, and Tables**: Using various HTML elements to organize content effectively.
- **HTML Forms**: Creating forms for user interaction in the contact section.

## Instructions Followed

1. **Navigation Bar**: Created a navbar with links to the following sections:
   - Home
   - About
   - Work
   - Resume
   - Contact

2. **Sections Created**:
   - **About Section**: A brief introduction about myself.
   - **Work Section**: Overview of my projects and experiences.
   - **Resume Section**: Displaying my professional qualifications.
   - **Contact Section**: An HTML form for visitors to reach out.

3. **Page Footer**: Added a footer to conclude the page, including links and copyright information.
